実際に見ていきましょう

array_change_key_case

  • キーを小文字か大文字に統一した配列を返す
  • 統一した結果同じkeyだったら後勝ちで上書き

array_chunk

  • 一次元配列を指定サイズ毎の二次元配列にして返す

array_column

  • 二次元配列から指定したキーの値だけを集めた配列を返す
  • indexに使う値を別キー指定で取ってくることも出来る
  • PHP5.5.0

array_combine

  • キー用配列と値用配列から配列を生成して返す

array_count_values

  • 同じ値が何個あるか集計して結果を配列で返す

array_diff_assoc

  • 一つ目の配列を元に二つ目の配列に対して比較
  • キーと値の組み合わせで比較
  • 見つかった差分を配列で返す

array_diff_key

  • 一つ目の配列を元に二つ目の配列に対して比較
  • キーで比較
  • 見つかった差分を配列で返す

array_diff_uassoc

  • 一つ目の配列を元に二つ目の配列に対して比較
  • キーと値の組み合わせで比較
  • 見つかった差分を配列で返す
  • 比較はユーザー定義関数を使う

array_diff_ukey

  • 一つ目の配列を元に二つ目の配列に対して比較
  • キーで比較
  • 見つかった差分を配列で返す
  • 比較はユーザー定義関数を使う

array_diff

  • 一つ目の配列を元に二つ目の配列に対して比較
  • 値で比較
  • 見つかった差分を配列で返す

小休止

array_fill_keys

  • 一つ目の配列を元に二つ目の配列に対して比較
  • 値で比較
  • 見つかった差分を配列で返す
